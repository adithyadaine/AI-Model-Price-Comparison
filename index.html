About LLM Ledger

Compare AI model pricing, performance, and capabilities in one place.

View up-to-date input and output costs per 1 million tokens for all major LLMs. No hidden fees or complex calculations.

Visual Analytics

Use interactive charts to visualize price trends and compare models side-by-side.

Benchmark API Access

To view advanced "Intelligence" and "Speed" metrics, this app connects to the Artificial Analysis API.

Why is an API key required?

High-quality benchmark data is expensive to produce. To keep this project open-source and free, we use a "Bring Your Own Key" model. You can get a free key (1000 requests/day) from Artificial Analysis and enter it via the API Key button in the sidebar.

How to Use
  1. Select Models: Use the sidebar to choose specific models or providers you want to compare.
  2. Analyze: Switch between Table, Bar Chart, Scatter Plot, and Timeline views using the top navigation.
  3. Filter: Use the price tier buttons (Low, Medium, High) to quickly narrow down options.
  4. Details: Click on any model name to view detailed specs including release date and specific provider info.

Designed for developers, researchers, and businesses to make informed decisions about AI infrastructure.

Privacy & Data

Your privacy is neither stored nor sold. LLM Ledger operates entirely on the client-side.

Zero Data Collection

We do not track your IP, search queries, or selected models. There is no user login and no server-side database storing your sessions.

Client-Side Processing

All interactions—sorting, filtering, and charting—happen instantly within your browser. Your preferences (like Theme) are saved only to your local device.

External Content

Model logos and data are fetched from trusted external sources (OpenRouter, Google Favicons). These requests are standard web traffic and do not transmit personal data.

Open Architecture

The application code runs visibly in your browser source. We prioritize transparency in both our data and our implementation.

Trust and security are core to the LLM Ledger experience.
